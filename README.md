# Auralis - Smart Contract Security Auditor

A comprehensive smart contract security auditing platform with AI-powered vulnerability detection.

## Project Structure

```
Auralis/
├── backend/          # FastAPI backend application
├── frontend/         # React frontend application
├── docs/            # Documentation
└── screenshots/     # Application screenshots
```

## Features

- Smart contract vulnerability detection
- AI-powered security analysis
- Real-time audit reports
- Multi-chain support

## Getting Started

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```
