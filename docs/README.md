# Auralis Documentation

## Overview
Auralis is a smart contract security auditing platform that helps developers identify vulnerabilities in their smart contracts.

## Architecture

### Backend (FastAPI)
- RESTful API for contract analysis
- AI-powered vulnerability detection
- Database for audit history

### Frontend (React)
- User-friendly interface
- Real-time audit results
- Dashboard for analytics

## API Documentation
API documentation is available at `/docs` when running the backend server.
