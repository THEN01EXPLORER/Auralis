### Long Description (500 words)

**Problem Statement:**
The blockchain industry faces a critical security crisis. In 2024 alone, over $2 billion was lost to smart contract vulnerabilities. Traditional security audits are expensive ($10K-$50K per contract), time-consuming (weeks to complete), and often inaccessible to independent developers. Existing automated tools suffer from high false-positive rates and lack intelligent context-aware analysis. Developers need real-time, AI-powered security feedback integrated into their workflow.

**Solution:**
GuardianAI Audit revolutionizes smart contract security through AI-powered vulnerability detection. Built with Amazon Q and powered by AWS Bedrock's Claude 3 Sonnet, our platform provides instant, intelligent security analysis for Solidity smart contracts.

**Key Features:**
- **6 Vulnerability Types Detected:** Re-entrancy attacks, integer overflow/underflow, access control violations, unchecked return values, timestamp dependence, and delegatecall injection
- **AI-Powered Analysis:** AWS Bedrock's Claude 3 Sonnet provides context-aware vulnerability detection with natural language explanations
- **Confidence Scoring:** Each detection includes a confidence score (0-100%) based on pattern strength and context
- **Visual Risk Assessment:** Real-time risk meter (0-100 scale) with color-coded severity indicators
- **Fix Recommendations:** AI-generated remediation suggestions with code examples
- **Professional UI:** Syntax-highlighted code editor, expandable vulnerability details, and responsive design

**Technical Architecture:**
- **Backend:** FastAPI with Pydantic validation, deployed on AWS Lambda
- **Frontend:** React 18 with CodeMirror editor, hosted on AWS Amplify
- **AI Engine:** AWS Bedrock (Claude 3 Sonnet) for intelligent analysis
- **API Gateway:** RESTful API with CORS support
- **Testing:** Comprehensive test suite with 100% backend coverage

**Amazon Q Integration:**
This entire project was built with Amazon Q as my AI development partner, demonstrating the power of AI-assisted development:
