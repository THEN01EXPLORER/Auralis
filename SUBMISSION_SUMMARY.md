## ✨ Key Features

1. **20 Vulnerability Detection Patterns**
   - Reentrancy, Integer Overflow, Front-Running, Flash Loans, etc.

2. **AI-Powered Analysis**
   - Semantic understanding beyond pattern matching
   - Context-aware vulnerability explanations

3. **GitHub Repository Scanner**
   - Analyze entire repos with one click
   - Multi-file tabbed interface

4. **Smart Remediation**
   - AI-generated fix explanations
   - Working code examples

5. **Export Reports**
   - JSON, Markdown, and Plain Text formats

6. **Professional UI**
   - Dark/Light theme toggle
   - Sample contracts dropdown
   - Keyboard shortcuts (Ctrl+Enter)
