## ğŸ’¡ AMAZON Q BEST PRACTICES (Learned)

### Effective Prompting
1. **Be specific:** "Create FastAPI endpoint with CORS" vs "Make API"
2. **Provide context:** Mention tech stack and requirements
3. **Request minimal code:** Ask for focused, essential code only
4. **Iterate:** Start simple, then enhance

### Code Quality
1. **Ask for type safety:** Request Pydantic models, TypeScript
2. **Request error handling:** Always ask for try-catch blocks
3. **Seek best practices:** Ask "What's the best way to..."
4. **Validate suggestions:** Review and test all generated code

### Documentation
1. **Generate as you go:** Don't wait until end
2. **Request structure:** Ask for organized, formatted docs
3. **Include examples:** Request code samples in docs
4. **Keep updated:** Regenerate docs when code changes
